# Dockerized Spring Jakarta Example Website
## Technology
* Java 21
* Maven
* Docker
* Postgres
## Framework
* Spring Boot
* Spring Security
* Spring Data JPA
* Spring Actuator
* Primefaces
* Lombok
* Test container
* Selenium
## Application
The Application is secured with Spring Security and provides a public endpoint for the health check.
### Startup
To Start the Application execute `start.sh`.
### Shutdown
To stop the application execute `stop.sh`
### Links
* [Application](http://localhost:8080)
